# Sudoku

2018 年秋季学期《软件工程基础》课程个人项目作业

### 目录结构

```
/ bin					可执行文件等二进制文件目录
	/ sudoku.exe				可执行文件
/ console				控制台程序工程目录
	/ generator.cpp				生成器实现
	/ generator.h				生成器头文件
	/ solver.cpp				求解器实现
	/ solver.hpp				求解器头文件
	/ sudoku.cpp				主函数实现文件
	/ ...					其他
/ UnitTest				控制台程序单元测试工程目录
	/ UnitTest.cpp				单元测试实现
	/ ...					其他
/ blog					博客作业目录
	/ img					Markdown 文件引用的图片目录
	/ blog.md				Markdown 格式的博客
```
